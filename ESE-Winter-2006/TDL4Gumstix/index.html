TDL4Gumstix is a case study showing how to extend the TDL Tool Chain to support a new platform. The platform chosen is Gumstix, a Linux-based computer system of the size of a gum stick. On this platform, real time tasks designed with the Timing Definition Language together with MATLAB Simulink are executed. A simple Java client simulates sensors and actuators that are used by the tasks running on the new platform. The communication between the client and the platform is done via UDP.
