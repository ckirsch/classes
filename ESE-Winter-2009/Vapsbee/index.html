Validation of an alternative positioning system in a real-time environment based on IEEE 802.15.4 (Zigbee) beacon

Vapsbee is a Zigbee based real-time positioning system. The setup mainly consists out of beacons planted into an environment and a moving receiver. The receiver calculates the position based on Received Signal Strength Indication (RSSI) and Link Quality Indicator (LQI) values from the IEEE 802.15.4 Stack.
