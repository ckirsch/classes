Project Outline

We want to implement an agent-system on the telos motes. On each mote, there will run an agent-interpreter (maybe a virtual machine). The whole system should work with (high) dynamic "mote networks". In order to achieve this, we need and want to implement the following "features":

  • Discovery Protocol
  • Reliable/fault-tolerant/at-least-once semantic Transfer Protocol
  • Routing Algorithm
  • Agents
  • Agent-Interpreter
  • Capability Model

The agents are using high level commands with the purpose to acquire data and provide them to the "master" (any computer connected to a mote).

Possible Applications:

  • Route Acquiring Agent (RAA)
  • Data Collecting Agent (DCA)
  • ...
  • Combinations of different Agents in one system
