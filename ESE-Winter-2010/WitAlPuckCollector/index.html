Project Description

The goal of our project is to construct and program a robot that is able to perform the task of collecting pucks. Puck collection is a discipline of the RobotChallenge.

Puck Collect

In the puck collect discipline two robots compete against each other on a square field. Goal of each robot is to collect as many pucks of its color as possible in a given time. A puck is counted as collected if it is on the robots home base.
